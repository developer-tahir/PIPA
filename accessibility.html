Accessibility Statement

Last updated: 10 May 2023

Our Commitment to Accessibility

PIPA Farm is committed to ensuring digital accessibility for people of all abilities, including those with dyslexia and other reading difficulties. We are continually improving the user experience for everyone, and applying the relevant accessibility standards.

Dyslexia-Friendly Features

We have implemented numerous features to make our application more accessible to users with dyslexia, which affects approximately 10% of Australians. Our dyslexia-friendly features include:

Dyslexia-Friendly Font

We use the Lexend font family, which was specifically designed to improve reading proficiency for those with dyslexia.

Low-Contrast Color Scheme

Our color scheme features subtle backgrounds and carefully selected text colors to reduce visual stress.

Enhanced Typography

We've implemented increased line spacing, letter spacing, and word spacing to improve text readability.

Dyslexia Mode

A dedicated dyslexia mode with further enhancements can be enabled in the settings menu.

Text Size Controls

Users can increase or decrease text size throughout the application to find their optimal reading size.

Line Length Optimization

Text containers are limited to 70 characters per line to prevent tracking issues.

Offline Support for Rural Areas

We understand that many Australian farms have limited internet connectivity. Our application includes:

  • Full offline functionality for core features
  • Background syncing when connection is restored
  • Clear indicators of connection status
  • Data-saving mode for low-bandwidth areas

Responsive Design for All Devices

PIPA Farm is designed to work on all devices commonly used in farm environments:

  • Desktop computers in the farm office
  • Tablets for field use
  • Mobile phones for on-the-go management
  • Optimized touch targets for use with gloves

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

PIPA Farm is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Accessibility Options in the Application

You can access the following accessibility options through the settings page:

Text Size
High Contrast
Reduced Motion

Keyboard Navigation

PIPA Farm supports full keyboard navigation. Common keyboard shortcuts include:

Tab Move to the next interactive element
Shift + Tab Move to the previous interactive element
Enter or Space Activate the current element
Esc Close popups or menus
Alt + 1 Go to Dashboard
/ Focus search box
